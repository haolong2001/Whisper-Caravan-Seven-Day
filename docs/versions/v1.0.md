# v1.0 Scope

## Intent

Package Whisper Caravan as a polished portfolio project with a clear product story, technical story, and demo narrative.

## Scope

- repository cleanup
- polished documentation
- demo assets
- architecture explanation
- deployable showcase if practical

## Required Assets

- clean README
- architecture diagram
- demo video
- live deployment if possible
- evaluation notebook
- short technical write-up
- interview demo script
- clear explanation of how RAG affects game state

## Success Criteria

- A reviewer can understand the product and technical value quickly.
- The demo is stable enough to show live.
- The project clearly communicates how retrieval changes gameplay outcomes.

## Non-Goals

- No requirement to turn the project into a commercial product
- No large live-service feature set
- No need for production-scale infrastructure unless it materially improves the portfolio story
