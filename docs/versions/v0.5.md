# v0.5 Scope

## Intent

Create a complete playable beta that feels like a coherent game slice rather than only a systems prototype.

## Scope

- two seven-day memory loops
- broader location coverage
- larger authored event set
- final Bear Court trial
- multiple endings

## Required Behavior

- Two seven-day memory loops.
- Four locations:
  - Deer Village
  - River Refugee Camp
  - Fox Market
  - Bear Court
- Six important NPCs.
- Twenty to thirty events.
- Main story built around the medicine conflict.
- Final Bear Court trial.
- Three to five endings.

## Final Trial Should Evaluate

- accepted evidence
- rejected evidence
- expired memories
- rumor conflicts
- faction reputation
- legal risk

## Success Criteria

- The game is playable end to end as a beta.
- Different routes lead to meaningfully different final outcomes.
- The trial uses the evidence system as a real payoff, not just a summary screen.

## Non-Goals

- No large-scale content generation
- No open-world map
- No combat system unless separately scoped
- No monetization systems
