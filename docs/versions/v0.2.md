# v0.2 Scope

## Intent

Expand the proof of concept into a playable seven-day loop where evidence accumulation and memory decay can be observed across multiple contexts.

## Scope

- Days 1-7 fully authored
- 4 choices per day
- faction and resource tracking
- multiple public evidence types: `record`, `song`, `rumor`
- Day 7 memory collapse preview
- Day 8 NPC reaction
- Bear Court evidence preview

## Success Criteria

- Different weekly routes produce different active evidence stacks on Day 8.
- The player can inspect which memories expired and which persisted.
- The Day 8 reaction is interpretable from the surviving evidence.
- The prototype feels like a coherent week-long system, not seven unrelated scenes.

## Out Of Scope

- persistent save data
- generalized NPC framework
- real backend retrieval
- full trial gameplay
- production balancing

## Notes

The current repository largely matches this scope already, and version metadata is now aligned to `0.2.0`.
