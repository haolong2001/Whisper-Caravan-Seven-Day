# v0.2 Scope

## Intent

Expand the proof of concept into a playable seven-day loop where evidence accumulation and memory decay can be observed across multiple contexts.

## Scope

- Days 1-7 fully authored
- 4 choices per day
- faction and resource tracking
- multiple public evidence types: `record`, `song`, `rumor`
- Day 7 memory collapse preview
- Day 8 NPC reaction
- Bear Court evidence preview

## Success Criteria

- [x] Different weekly routes produce different active evidence stacks on Day 8.
- [x] The player can inspect which memories expired and which persisted.
- [x] The Day 8 reaction is interpretable from the surviving evidence.
- [x] The prototype feels like a coherent week-long system, not seven unrelated scenes.
- [x] `npm run build` passes on June 26, 2026.
- [~] Manual in-browser smoke test is still recommended even though the code path is complete and build-verified.

## Out Of Scope

- persistent save data
- generalized NPC framework
- real backend retrieval
- full trial gameplay
- production balancing
