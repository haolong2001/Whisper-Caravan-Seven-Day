# v0.1 Scope

## Intent

Validate the core premise that delayed consequences can be modeled through memory persistence rather than immediate branching alone.

## Scope

- One main incident
- Limited number of generated memories
- Visual memory inspector
- Demonstration of seven-day forgetting
- One simple consequence readout

## Success Criteria

- The player can understand the difference between short-term and persistent traces.
- A single choice visibly produces downstream consequences later in time.
- The prototype is strong enough to justify expanding into a week-long loop.

## Out Of Scope

- Seven authored days
- faction economy depth
- court preview
- multiple evidence types beyond the initial baseline
- backend retrieval

## Notes

This version is best understood as the concept-validation milestone rather than the current code state.
