# Whisper Caravan 与流言同旅

![Whisper Caravan 初始界面](public/images/title-screen.png)

**Whisper Caravan 与流言同旅** 是一款关于记忆、流言、生存与审判的叙事选择游戏。

他们说，你做错了事。
你只有 **14 天**，然后熊法庭就会到来。

在旅途中，你会遇见鹿村、狐市、难民营、乌鸦经纪人与熊法庭。每天只有一个关键事件，每一次选择都会改变你遇见的人、留下的证据，以及 NPC 对你的态度。

但这个世界被诅咒了。

NPC 只能保留 **7 天** 的直接记忆。短暂的善意可能被遗忘，临时的证词可能失效。可记录、契约、流言与歌谣会继续存在，成为世界记住你的方式。

[开始游玩 Whisper Caravan](https://whisper-caravan-seven-day.vercel.app/)

## 游戏特色

* **14 天叙事流程**：每天一个关键事件
* **路线分歧**：不同选择会遇到不同事件与真相碎片
* **NPC 态度变化**：你的行为会影响信任、风险与结局
* **七日记忆机制**：短期记忆会消退，但不会彻底消失
* **证据系统**：记录、契约、流言与歌谣会以不同方式影响审判
* **多结局**：世界最终会记住哪一种版本，由你决定

## 本地运行

```bash
npm install
npm run dev
```

然后打开终端中显示的本地地址。
