# Whisper Caravan 与流言同旅

![Whisper Caravan 初始界面](public/images/title-screen.png)

**Whisper Caravan 与流言同旅** 是一款关于记忆、流言、生存与审判的叙事选择游戏。

他们说，你做错了事。
你只有 **14 天**，然后熊法庭就会到来。

在旅途中，你会遇见鹿村、狐市、难民营、乌鸦经纪人与熊法庭。每天只有一个关键事件，每一次选择都会改变你遇见的人、留下的证据，以及 NPC 对你的态度。

但这个世界被诅咒了。

NPC 只能保留 **7 天** 的直接记忆。短暂的善意可能被遗忘，临时的证词可能失效。可记录、契约、流言与歌谣会继续存在，成为世界记住你的方式。

## 游戏特色

* **14 天叙事流程**：每天一个关键事件
* **路线分歧**：不同选择会遇到不同事件与真相碎片
* **NPC 态度变化**：你的行为会影响信任、风险与结局
* **七日记忆机制**：短期记忆会消退，但不会彻底消失
* **证据系统**：记录、契约、流言与歌谣会以不同方式影响审判
* **多结局**：世界最终会记住哪一种版本，由你决定

## 架构与 AI 使用

Whisper Caravan 被设计为一个展示先进游戏 AI 整合的生产级方案：

* **RAG 驱动的 NPC 系统 (RAG-powered NPCs)**：采用异步混合 RAG 管道（结合 SQLite 与 Chroma 向量数据库）来动态查询并提取玩家的记忆、流言及证据。NPC 在第二周的响应会根据检索上下文的语义相关性、可信度与可见范围动态生成。
* **多智能体响应框架 (Multi-Agent Response Framework)**：在 LLM 对话生成与核心游戏状态引擎之间实现了严格的 Schema 验证和确定性边界约束。这在保持高质量生成式角色扮演的同时，保障了游戏循环（如信任增减、商铺价格变动、路线解锁等）不受 AI 幻觉的影响。

关于 AI 机制与系统架构的详细拆解，请参阅 [docs/GAME_AI_DESIGN.md](docs/GAME_AI_DESIGN.md)。
