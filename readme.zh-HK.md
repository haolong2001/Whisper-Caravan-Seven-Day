# Whisper Caravan 與流言同旅

![Whisper Caravan 初始畫面](public/images/title-screen.png)

**Whisper Caravan 與流言同旅** 是一款關於記憶、流言、生存與審判的敘事選擇遊戲。

他們說，你做錯了事。
你只有 **14 天**，然後熊法庭就會到來。

在旅途中，你會遇見鹿村、狐市、難民營、烏鴉經紀人與熊法庭。每天只有一個關鍵事件，每一次選擇都會改變你遇見的人、留下的證據，以及 NPC 對你的態度。

但這個世界被詛咒了。

NPC 只能保留 **7 天** 的直接記憶。短暫的善意可能被遺忘，臨時的證詞可能失效。可記錄、契約、流言與歌謠會繼續存在，成為世界記住你的方式。

## 線上遊玩

[開始遊玩 Whisper Caravan](https://whisper-caravan-seven-day.vercel.app/)

## 遊戲特色

* **14 天敘事流程**：每天一個關鍵事件
* **路線分歧**：不同選擇會遇到不同事件與真相碎片
* **NPC 態度變化**：你的行為會影響信任、風險與結局
* **七日記憶機制**：短期記憶會消退，但不會徹底消失
* **證據系統**：記錄、契約、流言與歌謠會以不同方式影響審判
* **多結局**：世界最終會記住哪一種版本，由你決定

## 本地運行

```bash
npm run dev
```
